# 🤖 AI Chatbot using Python

## 📌 Description
This project is a simple rule-based AI chatbot developed using Python. It uses basic Natural Language Processing (NLP) techniques to understand and respond to user inputs.

## 🚀 Features
- Responds to user queries
- Keyword-based matching
- Beginner-friendly AI project

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Download the file
2. Run `python chatbot.py`
3. Start chatting!
